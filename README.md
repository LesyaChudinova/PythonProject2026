# Анализ востребованности товаров на WildBerries
Авторы: Обухова Анна и чудинова Олеся, РИС-24-1

Проект выполнен в рамках курса «Анализ данных на Python».  
Цель — выявить текстовые и ценовые факторы, влияющие на востребованность товаров на маркетплейсе WildBerries.

Ссылка на датасет:
[Kaggle: Карточки товаров с Wildberries с метриками](https://www.kaggle.com/datasets/lchudinova/wildberries/data)


## Основные выводы

1. Читаемость — главный текстовый фактор
Корреляция с востребованностью: r = 0.18.  
Чем понятнее описание, тем успешнее товар.

2. Сенсорная компенсация
Корреляция между ценой и сенсорной лексикой: r = -0.36.  
Дешёвые товары активно используют слова, связанные с ощущениями (вкус, запах, текстура).

3. Манипуляции в названии — минус
Корреляция манипулятивной лексики с востребованностью: r = -0.06.  
Слова «скидка», «акция», «успей» в заголовке слегка снижают успех.

4. Кластеризация (k=5)
Выделено 5 ценовых сегментов с разными текстовыми стратегиями:
- Средний сегмент
- Бюджетный масс-маркет (самый большой)
- Сомнительные распродажи (низкий рейтинг)
- Премиум
- Высокое качество, малый охват (рейтинг 4.94)

5. Гендерные различия
Женские описания — длиннее, эмоциональнее, сенсорнее.  
Мужские — конкретнее, прагматичнее, с акцентом на статус.

Использованные методы
- Корреляционный анализ (Спирмен)
- ANOVA, t-тест
- Логистическая регрессия (AUC = 0.642)
- Кластеризация KMeans (k=5)
- NLP-фичи: плотность лексики, читаемость, конкретность
